Research Question:

How can we provide students with temporary relief who suffer from situational depression?

Pitch:

'PAWS FOR RELIEF' is a mobile app that provides comfort to those with situational depression by connecting them with one-on-one animal-assisted therapy.

This project was for HCDE 518: User Centered Design.

Role:

User experience designer, Researcher, Developer. I was not the visual designer.

Research:

  • Photo Diary
  • Competitive Analysis
  • Research into Animal Assisted Therapy & Depression

Methods:

  • User Centered Design
  • User needs
  • User scenarios
