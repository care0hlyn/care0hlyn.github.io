Tamagotchi Boyfriend

About:

'Tamagotchi Boyfriend' is a virtual reality app using leap motion and Unity. This project was built during a mini day-long VR hackathon in Seattle, WA. This game is based off the old 90s' game, Tamagotchi/Nano Pets, where instead of a pet, you keep your "boyfriend" happy by "feeding" him. (More like smashing food in his general direction and leveling up in XP).

We were a team of five people. My role was pair programming the prototype in Unity.

This was my first project for virtual reality!



This project is currently on-going and we aren't releasing the code or assets for this yet. If you'd like to know more about this project, feel free to ping me!
